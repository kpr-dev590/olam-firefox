
# Olam Malayalam Dictionary Firefox Extension

A Firefox extension that integrates the Olam Malayalam dictionary for quick and easy word lookups.

## Features
* Quick search for Malayalam meanings.
* Lightweight and fast popup interface.
* Works directly within the browser.

## Installation
1. Clone this repository.
2. Open Firefox and go to `about:debugging`.
3. Click "This Firefox" and "Load Temporary Add-on".
4. Select the `manifest.json` from this folder.
